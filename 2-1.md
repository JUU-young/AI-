## 리스트

#### 리스트의 개념 및 생성 방법
---
파이썬에서 리스트(list)는 여러개 의 값을 하나로 묶어서 저장하는 자료형
- 생성 방법: 리스트를 만들 때는 대괄호([])로 감싸주고 각 요솟값은 쉼표(,)로 구분해 준다.
```bash
리스트명 = [요소1, 요소2, 요소3, ...]
```
- 리스트의 생김새
```
a = []
b = [1, 2, 3]
c = ['life', 'is', 'very', 'beautiful']
d = [1, 2, 'life', 'is']
e = [1, 2, ['life', 'is']]
```
- 리스트에는 어떠한 자료형도 포함 할 수 있다.
- 리스트 저장방법
```bash
a = [0, 0, 0]
a[0] = 3
a[1] = 4
a[2] = 5
print(a)

```

- 리스트 사용 방법

| 각 변수 사용                    | 리스트 사용                |
| -------------------------- | --------------------- |
| a, b, c, d = 10, 20 30, 40 | aa = [10, 20 ,30 ,40] |
| a 사용                       | aa[0] 사용              |
| b 사용                       | aa[1] 사용              |
| c 사용                       | aa[2] 사용              |
| d 사용                       | aa[3] 사용              |
- 리스트는 각 인덱스의 시작이 0이다.
#### 빈 리스트의 생성과 항목 추가
----
- append 매서드를 사용한 항목 추가
```bash
aa = []
aa.append(0)
aa.append(0)
aa.append(0)
print(aa)
```
	출력결과
	[0,0,0,0]
- for 문을 활용한 항복 추가
```bash
aa = []
for i in range(0, 100):
  aa.append(0)
len(aa)
```
	출력 결과
	100
	> len()은 리스트의 항목의 개수를 세는 함수
	
#### 리스트 값에 접근하는 다양한 방법
---
1) 음수값으로 접근
```bash
aa = [10, 20, 30, 40]

print("aa[-1]은 %d, aa[-2]는 %d" % (aa[-1], aa[-2]))
```
	출력
	aa[-1]은 40, aa[-2]는 30
- 음수값으로 접근하면 뒤 항목부터 접근한다. aa는 aa[-4]까지 접근되므로 aa[-5]는 오류를 발생한다.

2) 리스트에 접근할 때 콜론(:)을 사용해 범위를 지정
```bash
aa = [10, 20, 30, 40]
aa[0:3]
aa[2:4]
```
	출력 결과
	[10, 20, 30]
	[30, 40]
3) 콜론의 앞이나 숫자 생략
```bash
aa = [10, 20, 30, 40]
aa[2:]
aa[:2]
```
	출력 결과
	[30, 40]
	[10, 20]
4) 리스트끼리 덧셈, 곱셈 연산
```bash
aa = [10, 20, 30]
bb = [40, 50, 60]
aa + bb
aa * 3
```
	출력 결과
	[10, 20, 30, 40, 50, 60]
	[10, 20, 30, 10,20,30, 10 ,20 ,30]
5) 리스트의 항목 건너뛰며 추출
```bash
aa = [1, 2, 3, 4, 5, 6, 7]
aa = [::2]
aa = [::-2]
aa = [::-1]
```
	출력 결과
	[1, 3, 5, 7]
	[7, 5, 3, 2]
	[7, 6, 5, 4, 3, 2, 1]
#### 리스트 값의 변경
---
1) 두 번째에 위치한 값을 1개 변경하는 방법
```
aa = [1, 2, 3]
a[1] = 200
a
```
	출력 결과
	[1, 200, 3]
2) 두 번째 값인 2를 200과 300이라는 2개의 값으로 변경
```bash
a = [1, 2, 3]
a[1:2] = [200, 300]
a
```
	출력 결과
	[1, 200, 300, 3]
3) 그냥 a[1] 사용시
```
a = [1, 2, 3]
a[1] = [200, 300]
a
```
	출력 결과
	[1, [200, 300], 3]

4) 리스트 값 삭제
```bash
a = [1, 2, 3, 4, 5]; del(a[1]); a
a = [1, 2, 3, 4, 5]; a[1:3] = []; a
```
	출력 결과
	[1, 3, 4, 5]
	[1, 5]
5) 리스트 자체 삭제
```bash
a =  [1, 2, 3];a = []; a
a =  [1, 2, 3];a = None; a
a =  [1, 2, 3];del(a); a
```
	출력 결과
	[]
	아무것도 안나옴
	오류 발생
#### 리스트 조작 함수
| 함수 | 설명 | 사용법 |
|---|---|---|
| append() | 리스트 맨 뒤에 항목을 추가한다. | 리스트명.append(값) |
| pop() | 리스트 맨 뒤의 항목을 빼낸다. (리스트에서 해당 항목이 삭제된다) | 리스트명.pop() |
| sort() | 리스트의 항목을 정렬한다. | 리스트명.sort() |
| reverse() | 리스트 항목의 순서를 역순으로 만든다. | 리스트명.reverse() |
| index() | 지정한 값을 찾아 해당 위치를 반환한다. | 리스트명.index(찾을값) |
| insert() | 지정된 위치에 값을 삽입한다. | 리스트명.insert(위치, 값) |
| remove() | 리스트에서 지정한 값을 삭제한다. 단, 값이 여러 개면 첫 번째 값만 지운다. | 리스트명.remove(지울값) |
| extend() | 리스트 뒤에 리스트를 추가한다. 리스트의 더하기(+) 연산과 기능이 동일하다. | 리스트명.extend(추가할리스트) |
| count() | 리스트에서 해당 값의 개수를 센다. | 리스트명.count(찾을값) |
| clear() | 리스트의 내용을 모두 지운다. | 리스트명.clear() |
| del() | 리스트에서 해당 위치의 항목을 삭제한다. | del(리스트명[위치]) |
| len() | 리스트에 포함된 전체 항목의 개수를 센다. | len(리스트명) |
| copy() | 리스트의 내용을 새로운 리스트에 복사한다. | 새리스트 = 리스트명.copy() |
| sorted() | 리스트의 항목을 정렬해서 새로운 리스트에 대입한다. | 새리스트 = sorted(리스트) |

## 튜플

- 순서대로 묶어서 저장하는 자료형
- 특징
	1. 변경 불가능: 한 번 만들면 값을 수정, 추가, 삭제할 수 없음-> 읽기만 가능해 읽기 전용 자료를 저장할 때 사용
	2. 순서가 있음: 인덱스로 접근 가능
	3. 서로 다른 자료형 저장 가능
#### 튜플의 생성
---
1) 튜플은 소괄호 ()로 생성
	- 튜플은 소괄호() 생략 가능, 항목이 하나인 튜플은 뒤에 쉼표(,)붙임
```bash
t1 = (10 , 20 ,30); t1
t2 = 10, 20, 30; t2
t3 = (10,); t3
t4 = 10,; t4
```
	출력
	 (10, 20, 30)
	 (10, 20, 30)
	 (10,)
	 (10,)
#### 튜플의 사용
---
기본적으로 튜플 값을 바꿀수 없다는 것을 제외하면 리스트 사용법과 거의 같다.
1) 튜플 범위에 접근
```bash
t1 = (10 , 20 ,30, 40)
t1[1:3]
t1[1:]

```
	출력
	 (20, 30)
	 (20, 30, 40)
	 (10,20, 30)

```
t1 = (10 , 20 )
```

	출력

## 딕셔너리(dictionary)
- 키(key) : 값(value) 쌍으로 데이터를 저장하는 자료형
- 이름표(키)로 값을 찾는 구조
#### 딕셔너리의 개념
---
1) 쌍 2개가 하나로 묶인 자료구조
	- 중괄호 {}로 묶어 구성, 키(key)와 값(value)의 쌍으로 구성
```bash
딕셔너리변수 = {키: 값1, 키2:값2, 키3: 값3, ...}
```
#### 딕셔너리의 생성
----
	- 키와 값은 사용자가 지정하는 것이지 규정은 없음
	- 주의점: 딕셔너리에는 순서가 없어 생성한 순서대로 딕셔너리가 구성되어 있다는 보장 없음
```bash
dic = {1:'a', 2:'b', 3:'c', 4:'d'}
dic
```
	출력 결과
	{1:'a', 2:'b', 3:'c', 4:'d'}
2) 딕셔너리 새로운 데이터 추가
```bash
dic = {1:'a', 2:'b', 3:'c', 4:'d'}
dic[5] = 'e' 
print(dic)
```
	출력 결과
	{1: 'a', 2: 'b', 3: 'c', 4: 'd', 5: 'e'}
3) 내부 데이터 삭제
```bash
dic = {1:'a', 2:'b', 3:'c', 4:'d'}
del(dic[2])
print(dic)
```
	출력 결과
	{1: 'a', 3: 'c', 4: 'd'}
4) 동일한 키를 가진 인덱스를 저장할 때 동일한 키를 갖는 딕셔너리를 생성하는 것이 아니라 마지막에 있는 키가 적용
```
dic = {1:'a', 2:'b', 3:'c', 4:'d', 1 : 'f'}
print(dic)
```
	{1: 'f', 2: 'b', 3: 'c', 4: 'd'}
#### 딕셔너리 접근
---
1) 키로 값에 접근
```python
student= {'학변' : 100, '이름': '강호동', '학과': '전기전자'}

print(student['학번'])
print(student['이름'])
print(student['학과'])
```
	출력 결과
	100
	강호동
	전기전자
2) 딕셔너리명.get(키) 함수를 사용해 키로 값에 접근
```python
student= {'학번' : 100, '이름': '강호동', '학과': '전기전자'}

print(student.get('이름'))
```
	출력 결과
	강호동
- 딕셔너리명[키]와 딕셔너리명.get(키)는 결과 같음
- 딕셔너리명[키]는 없는 키를 호출할 경우 오류가 발생하지만 딕셔너리명.get(키)는 없는 키를 호출하면 아무것도 반환하지 않음
3) 딕셔너리명.keys() 함수를 사용해 키로 값에 접근 : 딕셔너리의 키 값들 출력
```python
student= {'학번' : 100, '이름': '강호동', '학과': '전기전자'}

print(student.keys())
```
	출력 결과
	dict_keys(['학번', '이름', '학과'])
4) 딕셔너리명.items() 함수를 사용하면 튜플 형태로도 구할 수 있음

```python
student= {'학번' : 100, '이름': '강호동', '학과': '전기전자'}
print(student.items())
dict_items([('학번', 100), ('이름', '강호동'), ('학과', '전기전자')])
```


# 문자열

## 

#### 문자열의 개념
---
1) 리스트 코드와 비교
	- 리스트는 대괄호 []로 묶고 문자열은 작은 따옴표로 묶어 출력
```bash
s = "안녕하세요"

print(s[0])
print(s[1:3])
print(s[3:])
```
	출력결과
	안
	녕하
	세요
2) 더하기(+) 기호 사용해 연결, 곱하기(\*) 기호 사용 문자열 반복
```bash
s = '안녕' + '하세요'
print(s)
s = '안녕'*3
print(s)
```
	>>> 안녕하세요
	>>> 안녕안녕안녕
3) len() 함수: 리스트나 문자열의 개수를 셀 때 사용
```bash
s = '강원대학교'
print(len(s))
```
	>>> 5

## 객체 지향 언어
----
객체는 자체의 속성과 행동을 함께 가지는 독립적인 단위로, 현실 세계의 사물이나 개념을 소프트웨어적으로 모델링한 것이다.

간단하게 설명하면 자체적인 변수와 함수를 갖는 새로운 속성의 코드 정도로 이해해도 좋다.

### 등장 배경

소프트웨어가 점점 더 복잡해지고 크기가 커지면서 기존 절차지향언어로는 코드 관리와 유지보수가 어려워 졌다. 개발자들은 코드 재사용 및 모듈화를 통해 빠르고 정확한 개발을 원했다. 객체 지향 언어는 프로그램을 하나의 '흐름'이란 개념이 아닌 다수의 '객체'로 구성한다.

예를 들어, 건물의 위치와 기능, 관계도 등의 정보를 저장하여 지도로 시각화하는 프로그램을 개발한다고 가정할 때 만약 절차지향언어를 사용해서 하나의 흐름으로 프로그램을 개발한다면 코드가 너무 복잡해지고 새로운 건물이 생겨나는 등 프로그램을 수정할 때 매우 어려울 것이다.

객체지향언어라면 각 건물을 하나의 객체로 구현하고 다수의 객체를 동시에 원하는 만큼 사용하는 것으로 이를 구현할 수 있을 것이다. 만약 다른 지역의 동일한 프로그램을 사용하는 경우에도 해당 프로그램을 조금만 수정해서 구현할 수 있을것이다.

### 클래스(class)

파이썬에서 클래서는 서로 관련 있는 데이터와 기능을 하나로 묶어서 표현하는 설계도다.

 프로그램에서 어떤 대상을 표현하려면 그 대상이 가지는 데이터(속성)와 기능(동작)이 필요함. 이때 어떤 데이터가 있는지 어떤 기능을 하는지를 미리 정의해 놓은 것을 ==클래스==라고 한다. 이 클래스를 이용해 실제로 만들어진 것을 **객체(Object)**라고 한다.

- 특징: 속성과 기능을 함께 묶을 수 있다. 그래서 데이터와 그 데이터를 다루는 동작을 한 곳에서 관리할 수 있다. 하나의 클래스를 만들어 두면 그 구조를 바탕으로 여러 개의 객체를 만들 수 있다. 예시) 자동차 설계도가 있으면 그 설계도를 바탕으로 다른 자동차도 양산이 가능
- 장점: 코드를 체계적으로 관리가 가능해짐. 관련 있는 내용을 묶어 놓기 때문에 코드가 길어져도 구조를 파악하기 쉬워진다.

1. ==**클래스 정의**==

구성 요소:
- 필드(Field): 클래스 내부에 선언된 변수로, 객체의 상태 데이터를 저장한다.
- 매서드(Method): 클래스 내부에서 구현된 함수를 의미하며, 객체의 기능을 정의한다.

```bash
class Car :
	color = ""
	speed = 0
```
2. 클래스 변수 : 클래스 내부에 정의되어 해당 해당 클래스로 생성된 모든 인스턴스가 공동으로 사용하는 변수다. 메모리 공간이 클래스 차원에서 하나만 할당되므로 모든 객체가 값을 공유
```bash
myCar1 = Car()
myCar2 = Car()
```
3. 클래스 변수 활용
```bash
class Car:
    color = ""
    speed = 0
    count = 0   # 클래스 변수

    def __init__(self):
        self.speed = 0
        Car.count += 1


# 객체 생성
myCar1 = Car()
myCar1.speed = 30
print("자동차1의 속도는 %dkm이며, 현재 자동차 수는 %d대이다." % (myCar1.speed, Car.count))

myCar2 = Car()
myCar2.speed = 60
print("자동차2의 속도는 %dkm이며, 현재 자동차 수는 %d대이다." % (myCar2.speed, Car.count))
```
	>>> 자동차1의 속도는 30km이며, 현재 자동차 수는 1대이다.
	>>> 자동차2의 속도는 60km이며, 현재 자동차 수는 2대이다.
