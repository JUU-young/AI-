## 변수 선언
---
- C언어에서는 변수는 반드시 데이터 타입과 길이를 명시해야한다.
- Python에서는 변수를 선언할 때 데이터 타입이나 길이를 명시할 필요가 없으며, 이를 동적 타이핑이라고 한다.

>*c언어 변수 선언 방식*
```c
int a =3;
float b = 3.14;
char c[100] = "안녕";
```
>*파이썬 변수 선언 방식*
```python
a = 3
b = 3.14
c = '안녕'
```
#### python 문법

- C언어에서는 코드 한 줄의 종료를 세미콜론(;)으로 표기하지만, Python에서는 줄바꿈(Enter)로 이를 표기한다. 
- 또한 반복문&조건문의 블록을 작성할 때 C언어sms 중활호({})를 사용하여 표기하지만, Python에서는 들여쓰기(tab 또는 스페이스바)로 이를 표기한다.

>*C언어*
```c
int main(){
int i = 3;

if (i == 3){
	printf("i는 3입니다.\n");
	while (i > 7){
printf("i는 %d", i)}
return 0;
	}
}
```

>*파이썬*
```python
i = 3
if i == 3;
	print("i는 3입니다. \n");
	
	for i in range(5);
		print("i는 %d" &i)
```

## 자료 구조
---

#### 기본 자료형

- 파이썬에서 단일 값을 갖는 자료형을 기본 자료형이라고 하며, 음수와 양수의 구분이 없다.
- C언어에서는 문자와 문자열이 다른 자료형이었지만, 파이썬에서는 모두 문자열 취급한다.
- C언어와 동일하게 = 기호를 활용하여 대입이 가능하다.

```python
int_type = 10
float_type = 3.14
bool_Type = True
str_type = "안녕하세요"
None_Type = None
```

| 자료형         |         | 설명                                               |
| ----------- | ------- | ------------------------------------------------ |
| `int`       | 정수형     | 1, -1, 4 같은 정수를 표현하는데 사용                         |
| `float`     | 실수형     | 3.14, -0.13같은 실수 표현                              |
| `bool`      | Boolean | 참/거짓을 표현하기 위한 자료형으로 True/False 값만 들어갈 수 있음       |
| `str`       | 문자열     | `"Hello", "Hi"`와 같은 문자열, `"", ''`으로 문자열을 선언 가능하다 |
| `none_Type` | None    | 변수의 초기값이나 함수에서 반환할 값이 없는 경우 사용되는 특별 상수           |
#### 복합 자료형

- C언어에서의 배열처럼 Python에서도 여러 값을 저장할 수 있는 자료형이 있다. 이를 복합 자료형이라고 한다.
- list, tuple, dict 등이 있다.
- list는 `[]` 기호를 사용하여 선언할 수 있으며, 빈 리스트를 생성하는 것도 가능하다. C언어와 달리 서로 다른 타입의 데이터들을 하나의 리스트에 저장하는 것도 가능하다.
```python
list_type1 = []
list_type2 = [1, 2, 3]
list_type3 = [1, 3.14, 'a']
```

#### ==input 함수==
- input은 사용자가 키보드로 입력한 모든 것을 문자열로 저장한다.
```python
a = input()
>>>Life is too short, you need python
a
>>>Life is too short, you need python
```
- 프롬프트를 띄워 사용자 입력 받기사용자에게 입력 받을 때 '숫자를 입력하세요'나 '이름을 입력하세요'처럼 안내 무누를 보여 주고 싶을 때가 있을때 `input()`의 괄호 안에 안내 문구를 입력하면 된다.
```
number = input("숫자를 입력하세요: ")
```
- 괄호 안에 입력한 문구가 프롬프트로 나타난다.
- 여기서 주의 할 부분은 `input`은 입력되는 모든 것을 문자열로 취급한다. 따라서 number는 숫자가 아니라 문자열이다.

**입력 값을 숫자로 변환**
- input 함수는 항상 문자열을 반환하므로, 숫자 계산을 하려면 적절한 자료형으로 변환해야 한다.
```python
#정수 변환
age = input("나이를 입력하세요")
age = int(age)
# 실수 변환
height = input("키를 입력하세요(cm): ")
float(height)
# 한번에 변환
age = int(input("나이를 입력하세요: "))
```
- 자료형 형태 변환

|함수|변환 결과|예시|
|---|---|---|
|`int()`|정수|`int("123") → 123`|
|`float()`|실수|`float("3.14") → 3.14`|
|`str()`|문자열|`str(100) → "100"`|
|`bool()`|참/거짓|`bool(1) → True`, `bool(0) → False`|
|`list()`|리스트|`list("abc") → ['a', 'b', 'c']`|
|`tuple()`|튜플|`tuple([1, 2]) → (1, 2)`|
|`set()`|집합|`set([1, 1, 2]) → {1, 2}`|
|`dict()`|딕셔너리|`dict([("a", 1), ("b", 2)]) → {'a': 1, 'b': 2}`|

```python
print(1,2,3, end= ' and ')
print(4, 5, 6)
>1 2 3 and 4 5 6
```

---
산술 연산자의 종류
```python
result = 15
(result > 10) and (result<200)
```

| 연산자 | 의미     | 사용예        | 설명                           |
| --- | ------ | ---------- | ---------------------------- |
| =   | 대입 연산자 | a= 3       | 정수 3을a에 대입                   |
| +   | 더하기    | a = 3 + 4  | 3과 4를 더한 값을 a에  대입           |

## 제어문 (Control Statements)
---
단순히 자료구조와 기본 연산자로는 복잡한 프로그램을 구현할 수 없다. 개바자들은 여러 연산의 **반복, 조건 처리, 예외처리** 등을 사용하여 복잡한 프로그램을 구현한다.

제어문은 변수 선언, 연산 같은 일반 실행문과 달리 코드의 실행 순서를 바꿀 수 있다. 파이썬에서 제어문은 Block으로 구성되고 `:`와 `Tab` 키를 사용한 들여쓰기로 하나의 `Block`을 묶을 수 있다.

```python
if ... :
	Block 1의 코드 1
	Block 1의 코드 2
else ... :
	Block 2의 코드 1
	Blcok 2의 코드 2
	if ...:
		Block 2에 포함된 또 다른 BLock 2-1의 코드 1
		Block 2에 포함된 또 다른 Block 2-1의 코드 2
```

### ■ 중첩 for 문

하나의 for문안에 또 하나의 for문을 사용한다. 
- 예를 들어 위의 그림처럼 바깥쪽이 3번 도는 동안 안쪽은 각각 2번씩 돌아 총 3x2=6번 반복한다.
***중첩 for문의 기본형식
```bash
for i in range(0, 3, 1):
	for k in range(0, 2, 1):
	print("i값 : %d, k값 :%d" % (i, k))
```
	 i값 : 0, k값 : 0
	 i값 : 0, k값 : 1
	 i값 : 1, k값 : 0
	 i값 : 1, k값 : 1
	 i값 : 2, k값 : 0
	 i값 : 2, k값 : 1

- 처리 순서
   >외부 변수인 i는 0,1,2로 변경된 후 끝나지만, 내부 변수인 k는 0과 1을 계속 반복

#### 중첩 for문 예제
---
```bash
for i in ranege(2, 10, 1):
	for k in range(1, 10, 1):
		print("%d x %d = %2d" % (i, k, i*k))
	print("")
```
	출력결과
	2 x 1 = 2
	2 x 2 = 4
	2 x 3 = 6
	---중략---
	9 x 9 =81

## while 문

#### while 문의 기본 구조
---
```bash
while 조건문:
    수행할_문장1
    수행할_문장2
    수행할_문장3
    ...
```
- while 문은 조건문이 참일 동안 아래 문장들을 반복해서 수행
- 예)
```bash
age = 0
while age < 20:
	age = age + 1
	print("제 나이는 %d입니다." %age)
```
	제 나이는 1살입니다.
	제 나이는 2살입니다.
	제 나이는 3살입니다.
	제 나이는 4살입니다.
	...생략...
	제 나이는 20살입니다.
- 예시를 통해서 알 수 있듯이 age가 20보다 작은 동안 age에 1을 더하면서 구문을 반복한다.

#### 무한 루프
---
무한 루프란 무한히 코드를 반복한다는 의미다. 파이썬에서는 while문을 사용해 무한 루프를 구현할 수 있다.
- 무한 루프 적용: while 조건식 : 에 들어가는 조건식을 True로 지정
```bash
while True: 
    수행할_문장1 
    수행할_문장2
    ...
```
- 무한 루프의 예
```bash
>>> while True:
...     print("Ctrl+C를 눌러야 while문을 빠져나갈 수 있습니다.")
```
	출력결과
	Ctrl+C를 눌러야 while문을 빠져나갈 수 있습니다.
	Ctrl+C를 눌러야 while문을 빠져나갈 수 있습니다.
	Ctrl+C를 눌러야 while문을 빠져나갈 수 있습니다.
- 위 문장은 영원히 출력한다. [ctrl+c]를 눌러 while문을 빠져나갈 수 있다.
#### whlie-else문
---
파이썬의 while문은 else절을 사용 가능하다. whlil문이 정상적으로 종료 되었을 때 (break로 빠져나가지 않았을 때) else절이 실행된다
```bash
num = 0
while num < 2:
	print(num)
	num += 1
else:
	print("while 문이 정상 종료되었습니다.")
```
	출력
	0
	1
	while 문이 정상 종료되었습니다.
#### for 문과 while 문 비교
---
for 문은 반복할 횟수를 range() 함수에서 결정 후 그 횟수만큼 반복, while 문은 반복 횟수를 결정하기 보다는 조건식이 참일 때 반복하는 방식
- for 문과 비슷하게 사용할 수 있는 while문의 형식
```bash
변수 = 시작값
while 변수 < 끝값 :
	이 부분을 반복
	변수 = 변수 + 증가값
```

- for문으로 문장을 3회 출력하는 코드
```bash
for i in range(0, 3, 1) :
	print("%d: 안녕하세요?" %i)
```

- while 문으로 문장을 3회 출력하는 코드
```
i = 0
while i < 3:
	print("%d : 안녕하세요?", %i)
	i += 1
```

 #### break문
---
while문은 조건문이 참인 동안 반복해서 수행한다. 하지만 강제로 while문을 빠져나가야 하는 경우가 자주 발생한다. 이런 경우 break문을 사용한다.
- braek: 조건과 상관 없이 반복문을 바로 빠져나감. 반복이 끝나지 않아도 강제로 종료
``` bash
for i in range(1, 11):
    if i == 5:
        break
    print(i)
```
	실행결과
	1
	2
	3
	4
	5
- i == 5가 되는 순간 break 실행해 반복문을 종료한다.
- break문은 for문에서 동일하게 사용 가능하다.

#### continue 문
---
파이썬 프로그램이 실행되다가 countnue를 만나면 그 아래의 코드를 수행하지 않고 whlie 문의 조건을 판단하는 곳으로 점프하게 된다.
```
hap ,i =0, 0

for i in range(1, 101) :
	if i % 3 == 0:
		continue
	
	hap += i
print("1~100의 합계(3의 배수 제외) :", hap)
```
	출력 결과
	1~100의 합계(3의 배수 제외) : 3367

## 리스트

#### 리스트의 개념 및 생성 방법
---
파이썬에서 리스트(list)는 여러개 의 값을 하나로 묶어서 저장하는 자료형
- 생성 방법: 리스트를 만들 때는 대괄호([])로 감싸주고 각 요솟값은 쉼표(,)로 구분해 준다.
```bash
리스트명 = [요소1, 요소2, 요소3, ...]
```
- 리스트의 생김새
```
a = []
b = [1, 2, 3]
c = ['life', 'is', 'very', 'beautiful']
d = [1, 2, 'life', 'is']
e = [1, 2, ['life', 'is']]
```
- 리스트에는 어떠한 자료형도 포함 할 수 있다.
- 리스트 저장방법
```bash
a = [0, 0, 0]
a[0] = 3
a[1] = 4
a[2] = 5
print(a)

```

- 리스트 사용 방법

| 각 변수 사용                    | 리스트 사용                |
| -------------------------- | --------------------- |
| a, b, c, d = 10, 20 30, 40 | aa = [10, 20 ,30 ,40] |
| a 사용                       | aa[0] 사용              |
| b 사용                       | aa[1] 사용              |
| c 사용                       | aa[2] 사용              |
| d 사용                       | aa[3] 사용              |
- 리스트는 각 인덱스의 시작이 0이다.

#### 리스트 값에 접근하는 다양한 방법

	출력

	출력 결과

	[10, 20]
4) 리스트끼리 덧셈, 곱셈 연산

	출력 결과

5) 리스트의 항목 건너뛰며 추출

#### 리스트 조작 함수
| 함수        | 설명                                          | 사용법                 |
| --------- | ------------------------------------------- | ------------------- |
| append()  | 리스트 맨 뒤에 항목을 추가한다.                          | 리스트명.append(값)      |
| pop()     | 리스트 맨 뒤의 항목을 빼낸다. (리스트에서 해당 항목이 삭제된다)       | 리스트명.pop()          |
| sort()    | 리스트의 항목을 정렬한다.                              | 리스트명.sort()         |
| reverse() | 리스트 항목의 순서를 역순으로 만든다.                       | 리스트명.reverse()      |
| index()   | 지정한 값을 찾아 해당 위치를 반환한다.                      | 리스트명.index(찾을값)     |
| insert()  | 지정된 위치에 값을 삽입한다.                            | 리스트명.insert(위치, 값)  |
| remove()  | 리스트에서 지정한 값을 삭제한다. 단, 값이 여러 개면 첫 번째 값만 지운다. | 리스트명.remove(지울값)    |
| extend()  | 리스트 뒤에 리스트를 추가한다. 리스트의 더하기(+) 연산과 기능이 동일하다. | 리스트명.extend(추가할리스트) |
| count()   | 리스트에서 해당 값의 개수를 센다.                         | 리스트명.count(찾을값)     |
| clear()   | 리스트의 내용을 모두 지운다.                            | 리스트명.clear()        |
| del()     | 리스트에서 해당 위치의 항목을 삭제한다.                      | del(리스트명[위치])       |
| len()     | 리스트에 포함된 전체 항목의 개수를 센다.                     | len(리스트명)           |
| copy()    | 리스트의 내용을 새로운 리스트에 복사한다.                     | 새리스트 = 리스트명.copy()  |
| sorted()  | 리스트의 항목을 정렬해서 새로운 리스트에 대입한다.                | 새리스트 = sorted(리스트)  |

## 딕셔너리(dictionary)
- 키(key) : 값(value) 쌍으로 데이터를 저장하는 자료형
- 이름표(키)로 값을 찾는 구조

#### 딕셔너리의 개념
---
1) 쌍 2개가 하나로 묶인 자료구조
	- 중괄호 {}로 묶어 구성, 키(key)와 값(value)의 쌍으로 구성
```bash
딕셔너리변수 = {키: 값1, 키2:값2, 키3: 값3, ...}
```

#### 딕셔너리의 생성
----
	- 키와 값은 사용자가 지정하는 것이지 규정은 없음
	- 주의점: 딕셔너리에는 순서가 없어 생성한 순서대로 딕셔너리가 구성되어 있다는 보장 없음
```bash
dic = {1:'a', 2:'b', 3:'c', 4:'d'}
dic
```
	출력 결과
	{1:'a', 2:'b', 3:'c', 4:'d'}

2) 딕셔너리 새로운 데이터 추가
```bash
dic = {1:'a', 2:'b', 3:'c', 4:'d'}
dic[5] = 'e' 
print(dic)
```
	출력 결과
	{1: 'a', 2: 'b', 3: 'c', 4: 'd', 5: 'e'}
3) 내부 데이터 삭제
```bash
dic = {1:'a', 2:'b', 3:'c', 4:'d'}
del(dic[2])
print(dic)
```
	출력 결과
	{1: 'a', 3: 'c', 4: 'd'}
4) 동일한 키를 가진 인덱스를 저장할 때 동일한 키를 갖는 딕셔너리를 생성하는 것이 아니라 마지막에 있는 키가 적용
```
dic = {1:'a', 2:'b', 3:'c', 4:'d', 1 : 'f'}
print(dic)
```
	{1: 'f', 2: 'b', 3: 'c', 4: 'd'}
#### 딕셔너리 접근
---
1) 키로 값에 접근
```bash
student= {'학변' : 100, '이름': '강호동', '학과': '전기전자'}

print(student['학번'])
print(student['이름'])
print(student['학과'])
```
	출력 결과
	100
	강호동
	전기전자
	
2) 딕셔너리명.get(키) 함수를 사용해 키로 값에 접근
```bash
student= {'학번' : 100, '이름': '강호동', '학과': '전기전자'}

print(student.get('이름'))
```
	출력 결과
	강호동
	
- `딕셔너리명[키]`와 딕셔너리명.get(키)는 결과 같음
- `딕셔너리명[키]`는 없는 키를 호출할 경우 오류가 발생하지만 딕셔너리명.get(키)는 없는 키를 호출하면 아무것도 반환하지 않음

3) 딕셔너리명.keys() 함수를 사용해 키로 값에 접근 : 딕셔너리의 키 값들 출력
```python
student= {'학번' : 100, '이름': '강호동', '학과': '전기전자'}

print(student.keys())
```
	출력 결과
	dict_keys(['학번', '이름', '학과'])
	
4) 딕셔너리명.items() 함수를 사용하면 튜플 형태로도 구할 수 있음
 ```python
student= {'학번' : 100, '이름': '강호동', '학과': '전기전자'}
print(student.items())
   
dict_items([('학번', 100), ('이름', '강호동'), ('학과', '전기전자')])
 ```
